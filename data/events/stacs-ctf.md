# Hack the Flag! 🚩

Test your cybersecurity skills in the **STACS CTF Competition**! Solve challenges across multiple categories and compete for amazing prizes.

## What is a CTF?

A **Capture The Flag (CTF)** competition is a cybersecurity challenge where participants solve puzzles to find hidden "flags". Think of it as a treasure hunt for hackers!

## Challenge Categories

### 🔐 Cryptography
Crack codes, decrypt messages, and break ciphers.

### 🌐 Web Exploitation
Find vulnerabilities in web applications and exploit them responsibly.

### 🔍 Forensics
Analyze files, network traffic, and memory dumps to find hidden information.

### ⚙️ Reverse Engineering
Understand how programs work by analyzing their code and behavior.

### 💻 Binary Exploitation
Find and exploit vulnerabilities in compiled programs.

### 🕵️ OSINT
Use open-source intelligence gathering techniques to solve challenges.

## Competition Format

- **Duration**: 24 hours
- **Team Size**: 1-4 people
- **Difficulty**: Challenges range from beginner to advanced
- **Scoring**: Points awarded based on challenge difficulty
- **Platform**: Online CTF platform (link provided upon registration)

## New to CTFs?

Don't worry! We'll have:
- A **beginners' workshop** at the start
- **Hints** available for tricky challenges
- **Mentors** on Discord to help
- **Writeups** published after the event

### Recommended Preparation
- Basic command line knowledge
- Familiarity with at least one programming language
- Curiosity and problem-solving mindset!

3. **Write-ups welcome** - but only after the competition ends
