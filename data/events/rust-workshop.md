Ever wanted to make something in C for that sweet native CPU speed — but kept getting segfaults?

You should learn Rust. Rust is the fastest-growing programming language ever: as fast as C, but without segfaults, data races, deadlocks, use-after-free bugs, buffer overflows, or null pointer errors.

If you want to learn how to write blazingly fast programs — or just want something nice on your CV — come to the STACS Dev Rust workshop.
