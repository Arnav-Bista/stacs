# Build Your First Website!

Join us for a **beginner-friendly workshop** where you'll learn the fundamentals of web development and build your first website from scratch!

## What You'll Learn

### HTML & CSS Basics
- Structure your web pages with HTML
- Style them with CSS
- Responsive design principles

### JavaScript Fundamentals
- Variables, functions, and control flow
- DOM manipulation
- Event handling

### Practical Project
- Build a personal portfolio website
- Deploy it online for free
- Take home your own website!

## Prerequisites

**No prior experience required!** This workshop is designed for complete beginners.

### What to Bring
- Laptop with a text editor installed (we recommend [VS Code](https://code.visualstudio.com/))
- Charger
- Enthusiasm to learn!
